# NerdyPress Release

Version: v0.1.0
Prerelease: false

## Change Notes

- Added the first public NerdyPress generator experience for building VitePress sites visually.
- Added live site preview with theme palettes, style presets, font pairings, button styles, and card styles.
- Added separate header logo, site logo, favicon, footer links, footer alignment, and social icon placement controls.
- Added Simple Icons support with selectable social icons and icon previews.
- Added homepage button and feature card customisation.
- Added a proper page editor with page list, slug/path controls, descriptions, markdown editing, and live page preview.
- Expanded plugins and integrations with local search, Mermaid, sitemap, auto sidebar, edit links, last updated metadata, PWA support, Open Graph tags, robots.txt, analytics, and deployment target options.
- Added generated deployment support for GitHub Pages, Cloudflare Pages, Netlify, and Vercel.
- Added NerdyPress branding, app links, favicon, and “Made by Nerdy Technician” credit.
- Added README, issue templates, pull request template, and release automation using this `RELEASE.md` file.
